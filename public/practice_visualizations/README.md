# Файлы визуализаций практик

Положите медиафайлы в эту папку. Пути относительно `app/src/main/assets/`.

## Стослоговая мантра (`hundred_syllable_mantra`)

```
practice_visualizations/hundred_syllable_mantra/visualization.mp4 — фиксированное видео
```

## Сале О (`SALE_O`)

```
practice_visualizations/sale_o/poster.png    — статичная схема
practice_visualizations/sale_o/visualization.mp4 — видео (без звука, зацикленное)
```

## Ма Три (`MA_TRI`)

```
practice_visualizations/ma_tri/poster.png              — одна схема для всех
practice_visualizations/ma_tri/male/visualization.mp4   — видео (мужской)
practice_visualizations/ma_tri/female/visualization.mp4 — видео (женский)
```

Пол практикующего в **Настройках** выбирает, какое видео показывать.

## Ду Три Су (`DU_TRI_SU`) — картинка зависит от пола в настройках

```
practice_visualizations/du_tri_su/male/poster.png
practice_visualizations/du_tri_su/female/poster.png
practice_visualizations/du_tri_su/visualization.mp4
```

## Рекомендации

- **MP4**: H.264, без аудио, 720p или меньше, короткий loop (5–15 с)
- **PNG**: для схем (poster), вертикальная или квадратная ориентация

Пол практикующего: **Настройки → Пол практикующего**.

Если файл отсутствует, соответствующий блок на экране практики не показывается.
